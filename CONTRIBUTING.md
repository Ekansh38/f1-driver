# Contributing

Bug reports and pull requests are welcome.

## How to contribute

1. Fork the repo and create a branch
2. Make your changes
3. Open a pull request with a short description of what you changed and why

## Guidelines

- Keep changes focused. One thing per PR.
- Test your changes locally before submitting (`python main.py`)
- If you're adding a track, include all four required files (`bg.png`, `track_mask.png`, `track_data.png`, `track.json`)
